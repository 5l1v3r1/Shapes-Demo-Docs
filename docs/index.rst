eProsima Shape Demo Analytics
=============================

This document presents a brief introduction to eProsima Shapes Demo application. This application can be used as an introduction to the power of eProsima Fast RTPS. 

Shapes Demo published and subscribers different moving shapes of different colors and sizes. Each shape is published in its own topic (Square, Triangle or Circle) and a single instance can publish or subscriber to multiple topics at the same time.
